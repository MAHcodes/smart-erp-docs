# smart-erp-docs

## how to build

1. clone the repo
```
git clone https://github.com/MAHcodes/smart-erp-docs
```

2. cd into the repo
```
cd smart-erp-docs
```

3. install dependencies
```
npm i
```

4. build
```
npm run build
```
this will generate an optimized production build under a new directory `./build/`

5. serve the production build
```
npm run serve
```
